gazetteers
==========

This repository contains various gazetteers (lists) that are of use in natural language processing.

The following gazetteers are included:

- Airport codes
- Airport names
- Colors
- Constellations
- US counties
- Countries
- World currencies
- Deserts
- Islands
- Length measurement units
- Numbers
- Numerals
- Ordinal numbers
- Planets
- Seas
- US States
- World cities
